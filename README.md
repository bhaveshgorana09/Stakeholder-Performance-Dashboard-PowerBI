# 📊 Stakeholder Performance Dashboard (Power BI)

## 🔍 Overview
The **Stakeholder Performance Dashboard** is an interactive business intelligence project built using Power BI. It provides a comprehensive analysis of stakeholder value creation, profitability, regional performance, and financial metrics such as Assets Under Management (AUM) and cost ratio.

The dashboard helps organizations monitor key performance indicators (KPIs), track trends, and make data-driven decisions efficiently.

---

## 🎯 Objectives
- Evaluate stakeholder profitability and financial contribution  
- Track performance across regions and tenure  
- Analyze AUM distribution and cost ratio vs profitability  
- Enable interactive and dynamic data exploration  

---

## 🛠️ Tools & Technologies
- **Power BI** – Data visualization and dashboard development  
- **Power Query (ETL)** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – KPI creation and calculations  
- **Excel / CSV** – Data storage and preprocessing  

---

## 🏗️ Dashboard Architecture

1. **Data Source**  
   - Stakeholder financial and regional datasets (Excel/CSV)

2. **ETL Process**  
   - Data cleaning and transformation using Power Query

3. **Data Modeling**  
   - Relationships between stakeholders, regions, tenure, and financial metrics

4. **Visualization Layer**  
   - Interactive dashboards with KPIs, charts, slicers, and filters

---

## 📊 Key Features
- Stakeholder profitability analysis  
- Region-wise and tenure-based performance tracking  
- AUM distribution visualization  
- Cost ratio vs profitability comparison  
- Interactive filters and drill-down capabilities
